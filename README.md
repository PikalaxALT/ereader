# Card eReader + (US)

This is a disassembly of the Card eReader + (US) ROM dump.

It builds the following ROM:

* ereader.gba `sha1: 4b7005e81ee9ffee333b3f6095c3611800b56588`

For instructions on how to set up this repository, see [INSTALL.md](INSTALL.md).
